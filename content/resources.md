<h3>Tools and Platforms</h3>

**Data Wrangling**

[Google Sheets](https://workspace.google.com/products/sheets/): Web-based spreadsheet application with data wrangling functionality.

[OpenRefine](https://openrefine.org/): A powerful free, open source tool for working with messy data.

**Geocoding**

[Geocodio](https://www.geocod.io/): A geocoding platform that utilizes tabular data. The platform works with coordinates, addresses, and place names in the United States and Canada.

[GPS Visualizer Geocoder](https://www.gpsvisualizer.com/geocoder/): This tool can provide coordinates throughout the world for current places and addresses.

**Mapping**

[ArcGIS Online](https://www.arcgis.com/): Users may first be overwhelmed by the numbers of features and possibilities, but dedicated learning can result in powerful and informative maps.

[Carto](https://carto.com/): GIS cloud service that allows for free access with an [educator account](https://docs.carto.com/faqs/carto-for-education#how-can-i-get-an-educator-account).

[Folium](https://python-visualization.github.io/folium/latest/index.html): Python library that generates leaflet maps with Python code.

[Google Earth](https://www.google.com/earth/about/versions/): A more robust version of Google My Maps with 3D views.

[Google My Maps](https://www.google.com/mymaps): A good way to get started with mapping. It allows for placing points manually, drawing polygons and lines, and stylizing what you map.

[kepler.gl](https://kepler.gl/): Open-source geospatial analysis tool for large-scale data sets.

[leaflet.js](https://leafletjs.com/): JavaScript library that allows users to produce lightweight, interactive maps.

[MapWarper](https://mapwarper.net/): A free to use, open-source map warper / map georectifier, and image georeferencer tool for individuals and small groups.

[Palladio](https://hdlab.stanford.edu/palladio/): A suite of visualization and analysis tools designed by scholars to ask and answer research questions.

[Re:Earth](https://reearth.io/): Free, open, and highly extensible webGIS platform.

[Tableau Public](https://public.tableau.com/app/discover): A free version of [Tableau](https://www.tableau.com/), a data visualization platform that allows for map making.
